# LangChain Components Example

A collection of practical examples and experiments using **LangChain** and **LangGraph** concepts including:

- Agents
- Memory
- State Management
- Parallelization
- Streaming
- Subgraphs
- Map Reduce
- Human-in-the-loop workflows
- Prompt Templates
- Tool Calling

This repository is designed for learning and understanding core LangChain/LangGraph building blocks through hands-on Jupyter notebooks.

---

# Repository Structure

| File | Description |
|---|---|
| `agents_langgraph.ipynb` | Working with LangGraph agents and state schemas |
| `breakpoint.ipynb` | Human-in-the-loop workflows using breakpoints and time travel |
| `long_short_memory.ipynb` | Understanding long-term and short-term memory handling |
| `map_reduce.ipynb` | Map Reduce workflow demo using Send |
| `parallelization.ipynb` | Running tasks in parallel using LangGraph concepts |
| `prompt_template.ipynb` | Prompt template examples and formatting |
| `states_and_memory_langgraph.ipynb` | State management and memory handling in LangGraph |
| `stream.ipynb` | Streaming outputs and streaming workflows |
| `subgraphs.ipynb` | Building and using subgraphs |
| `time_travel.ipynb` | Time travel debugging and workflow replay |
| `tool_toolkit_agents.ipynb` | Tool calling and toolkit-based agents |
| `working_with_text.ipynb` | Text processing and manipulation examples |
| `requirements.txt` | Project dependencies |

# Setup

Clone the repository:

```bash
git clone [<your-repository-url>](https://github.com/adityash97/DataScience.git)
cd langchain_components_example
```

Create virtual environment:

```bash
python -m venv venv
```

Activate virtual environment:

### Linux / Mac

```bash
source venv/bin/activate
```

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Start Jupyter Notebook:

```bash
jupyter notebook
```
